# Introdução a ETL 

Este repositório tem como objetivo apresentar os **conceitos fundamentais de ETL (Extract, Transform, Load)**, abordando o processo de extração, transformação e carga de dados. O conteúdo é voltado para estudantes, iniciantes em dados e profissionais que desejam compreender a base dos processos de integração e tratamento de dados.

---

## Objetivos

- Compreender o conceito de ETL  
- Entender as etapas de **Extração, Transformação e Carga**  
- Conhecer os principais casos de uso de ETL  
- Apresentar boas práticas no tratamento de dados  
- Introduzir ferramentas e tecnologias utilizadas em processos de ETL  

---

## O que é ETL?

ETL é um processo utilizado para **integrar dados provenientes de diferentes fontes**, realizando a extração dos dados brutos, sua transformação em um formato adequado e, por fim, o carregamento em um destino final, como um banco de dados ou um data warehouse.

---

## Etapas do Processo ETL

### 1️⃣ Extração (Extract)
Coleta de dados a partir de diferentes fontes, como:
- Bancos de dados
- APIs
- Arquivos CSV, JSON ou XML
- Sistemas legados

### 2️⃣ Transformação (Transform)
Processamento dos dados extraídos, incluindo:
- Limpeza de dados
- Padronização de formatos
- Tratamento de valores nulos
- Enriquecimento e agregações

### 3️⃣ Carga (Load)
Inserção dos dados transformados no destino final, como:
- Data Warehouse
- Data Lake
- Bancos de dados relacionais ou NoSQL
